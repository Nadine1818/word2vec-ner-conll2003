# Word2Vec + NER on CoNLL-2003

This repo contains the notebook **`word2vec_ner_conll2003.ipynb`**, which experiments with named entity recognition (NER) using Word2Vec-style embeddings on the **CoNLL-2003** dataset.

## Contents
- `word2vec_ner_conll2003.ipynb` — main notebook

## Running
1. Create an environment (optional but recommended)
2. Install typical NLP dependencies used in the notebook (e.g., `numpy`, `pandas`, `gensim`, `scikit-learn`, `torch`/`tensorflow` as applicable)
3. Open and run the notebook in Jupyter / VS Code

> Note: Dataset download/loading is handled inside the notebook (if applicable).
